# Como Compilar e Executar

### Pré-requisitos
* Java JDK (versão 8 ou superior) instalado e configurado no PATH.
* Os ficheiros de dados `collaboration_graph.txt` e `as_graph.txt` devem estar na mesma pasta que os ficheiros `.java`.

### Opção 1: Execução via Linha de Comando (Recomendado)

Navegue até a pasta do projeto e execute os seguintes comandos:

```bash
# 1. Compilar todos os ficheiros .java
javac *.java

# 2. Executar a classe principal
java LeitorDeGrafo
